### Install with brew:
```
# register apimation brew tap
brew tap apimation/brew-taps git@github.com:apimation/brew-taps.git

# update brew to update apimation brew tap
brew update

# install latest apimation version
brew install apimation
```

##### General help:
##
```
  apimation -h
```
